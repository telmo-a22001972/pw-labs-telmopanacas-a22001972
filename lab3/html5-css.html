HTML5

  • HTML5 (Hypertext Markup Language, atualmente em sua 5° versão) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software.

Elementos Semânticos HTML5 Elemento usado? Se usado, onde?
Header ✅ Todas as páginas no cabeçalho
Nav ✅ No menu de navegação de todas as páginas
Main ✅ Em todas as páginas depois do menu de navegação
Article ❌

Aside ❌

CSS

  • CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo (en-US) usada para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML). O CSS descreve como elementos são mostrados na tela, no papel, na fala ou em outras mídias.

Seletor CSS Foi usado? Se usado, onde?
Seletor Universal(*) ❌ O seletor universal (*) aplica o estilo a todos os elementos da página.
Seletor Tipo ✅ O seletor de tipo usa o nome do marcador a estilizar. A regra será aplicada a todos os elementos que possuam esse marcador.
Usado no info.html e html5-css.html.
Seletor de Classe ✅ O seletor de classe associa um nome a um estilo.
No seletor de classe, o nome (case-sensitive) é antecedido de .
Seletor de id ❌ O atributo id permite identificar univocamente um elemento do HTML
No seletor de id, o identificador é antecedido de #.
Selector de Atributo ❌ Seletores que se aplicam a atributos de elementos.
Pode ser apenas referido o atributo, e.g., [target].


Combinadores de seletores Foi usado? Se usado, onde?
div p ❌ Todos os p que estão “dentro” de um div.
div > p ❌ Todos os p que estão “logo dentro” de div.
div ~ p ❌ Todos os p que estão colocados depois de um div.
div + p ❌ Todos os p que estão colocados “imediatamente” depois de um div.
div,p{ color: red; } ❌ Os div e p ficacarão com fonte vermelha.
